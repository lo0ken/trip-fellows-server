# Trip Fellows Server
## Описание
Бэкенд для Android - приложения <b>Trip Fellows<b> ([GitHub](https://github.com/lo0ken/trip-fellows-client))
  
Реализовывает REST API
  
## Используемые технологии
* Java
* Spring (Spring Boot, Spring Data, Spring Security)
* Hibernate
* Flyway
* Postgres
* JUnit
* Mockito
* Lombok
* MapStruct
* EasyRandom
* Swagger
* Firebase
